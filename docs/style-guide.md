# Style Guide

## Typography
- Title: bold, 24–28px, high-contrast
- Subtitle: 14–16px, medium contrast
- Body: 14–16px

## Color Palette
- Primary: #7c3aed
- Background (light): #FFFFFF
- Background (dark): #0E1117
- Text (light): #0E1117
- Text (dark): #FAFAFA
- Panel (dark): #1F2937

## Spacing
- Container padding: 16–32px
- Component gap: 12–16px

## Components
- Panel: bordered, 12px radius, theme-aware background
- Pill: rounded, small padding, theme-aware text and border
- Circle: accent border, centered value

## Accessibility
- Contrast ratio AA+ on text and controls
- Keyboard focus outlines visible
- ARIA labels on section titles
