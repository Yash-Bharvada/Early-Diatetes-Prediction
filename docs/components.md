# Component Library

## Panel
Purpose: Group related inputs or results.
Usage: Render title + subtitle + content.

## Pill
Purpose: Display compact numeric or state values.
Usage: `value` rendered within rounded chip.

## Circle
Purpose: Visualize percentage prominently.
Usage: Centered percentage with accent border.

## Tabs
Purpose: Separate input from results.
Usage: `Input Data`, `Results`.

## Buttons
Primary: Calculate Risk Score.
Secondary: Modify inputs, Recalculate.
